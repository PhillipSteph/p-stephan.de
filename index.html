Game of Noah

Game of Noah, auch unter anderen Namen bekannt, ist ein Java-basiertes Videospiel, in dem es um Strategie und Kombat geht. Ziel ist es, einen möglichst hohen High-Score zu erzielen. Munition zum Schießen muss erstmal aufgesammelt werden. Pass aber auf, denn dein Magazin hat nur bedingt Platz. Außerdem werden Hindernisse platziert, durch Feuer, welches deine Schmerzens-Skala erhöht. Sehr viel Action auf einmal, doch wie weit schaffst du es? Die Steuerung ist recht simpel, jedoch empfiehlt es sich, eine Maus zu verwenden. WASD zum steuern, Maus-Links zum schießen.
Du bist interessiert, es auszuprobieren? kommst du zum Download!
Zum Spielen wird eine Java Runtime Environment benötigt. Falls diese vorhanden ist, muss das Spiel nur entpackt und danach ausgeführt werden. Entweder durchs öffnen oder über das Terminal mit java -jar ./game.jar. Dieses Spiel wurde bereits mal heruntergeladen!

Game of Physics

Game of Physics ist ebenfalls ein Java-basiertes Videospiel. Hierbei ist es aber auch als eine Simulation zu bezeichnen. Durch eine Simulation von verschiedenen Elementen wie Wasser, Sand, Wolken oder Gebirgen kann man die Kreativität in diesem Spiel den freien Lauf lassen. Die Steuerung kann ein wenig kompliziert wirken. Deshalb hier die wesentlichen Features: Durch links- und rechts-click der Maus können Blöcke platziert und abgebaut werden. Durch das simultane Betätigen der Shift-Taste wird ein quadratischer Bereich platziert. Die Enter-Taste ohne Shift ermöglicht es, einen Kreisförmigen-Bereich zu platzieren.
Probier die Simulation doch einfach mal selber aus. findest du den Download!
Zum Spielen wird eine Java Runtime Environment benötigt. Falls diese vorhanden ist, muss das Spiel nur ausgeführt werden. Entweder durchs öffnen oder über das Terminal mit java -jar ./Physics.jar. Dieses Spiel wurde bereits mal heruntergeladen!

Game of Blocks

Game of Blocks ist ein Sandbox-Spiel, welches auch mit Java programmiert wurde. Auch hier ist die Kreativität der größte Faktor des Spiels. Sammle Material, um eigene Kreationen zu erschaffen. Baue ein Haus, baue Personen oder kreiere dein eigenes Pixelart! Besonderheit dieses Spiels ist es, dass es im Gegensatz zu den anderen Spielen eine Multiplayer-Funktion anbietet. Kreiere Kunst, chatte mit der Chat-Funktion oder erkunde andere Bauten, zusammen mit Anderen! Platzieren und Löschen von Blöcken, schaffst du mit der Enter-, sowie der Backspace-Taste. Laufen mit WASD, Chatten mit Space, Rennen mit Shift und Schleichen mit Strg. Bist du interessiert? Dann probier es doch selber aus! findest du den Download!
Zum Spielen wird eine Java Runtime Environment benötigt. Falls diese vorhanden ist, muss das Spiel nur ausgeführt werden. Entweder durchs öffnen oder über das Terminal mit java -jar ./game.jar. Dieses Spiel wurde bereits mal heruntergeladen!
